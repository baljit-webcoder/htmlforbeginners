Advanced Multimedia in HTML


<picture>

The <picture> element helps display different image formats or sizes based on device or browser. You can give options like .webp, .jpg, etc.

<picture>
  <source srcset="image.webp" type="image/webp">
  <source srcset="image.jpg" type="image/jpeg">
  <img src="fallback.jpg" alt="Descriptive image text">
</picture>

<iframe>

The <iframe> element allows embedding another website, video, map, or even a PDF viewer inside your HTML.

<iframe src="https://example.com" width="600" height="400" title="Example Iframe"></iframe>

<source>

The <source> tag is used inside <video>, <audio>, or <picture> to offer format options.

<video controls>
  <source src="movie.mp4" type="video/mp4">
  <source src="movie.ogg" type="video/ogg">
</video>

<track>

The <track> tag adds subtitles, captions, or descriptions for videos. These files are usually .vtt files.

<video controls>
  <source src="movie.mp4" type="video/mp4">
  <track src="subtitles.vtt" kind="subtitles" srclang="en" label="English">
</video>

<canvas>

The <canvas> tag provides an area where JavaScript can draw shapes, games, or visual effects.

<canvas id="myCanvas" width="300" height="150"></canvas>

<script>
const canvas = document.getElementById("myCanvas");
const ctx = canvas.getContext("2d");
ctx.fillStyle = "lime";
ctx.fillRect(10, 10, 150, 100);
</script>

<object>

The <object> tag can embed PDFs, HTML files, images, or apps directly into your page.

<object data="document.pdf" type="application/pdf" width="600" height="400">
  Your browser doesn't support PDFs.
</object>

<embed>

The <embed> tag is a simpler version of <object>. It's used to embed media like videos, games, or documents.

<embed src="file.svg" type="image/svg+xml" width="400" height="300">
