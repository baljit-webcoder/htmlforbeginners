HTML Tables – Full Tutorial


This page helps you understand everything about HTML tables and how to style them using CSS.

📚 HTML Table Tags - Explained Clearly

<table> – Table Container

Defines the start and end of a table.

<table> ... </table>

<tr> – Table Row

Creates one horizontal row inside the table.

<tr>
  <td>A</td>
  <td>B</td>
</tr>

<td> – Table Data

Represents a single normal cell.

<td>Hello</td>

<th> – Table Header

Defines a heading cell. Bold and centered by default.

<th>Name</th>

<caption> – Table Title

Adds a title at the top of the table.

<caption>Student Info</caption>

<thead>, <tbody>, <tfoot>

Groups parts of the table: header, body, and footer.

<thead><tr><th>Col1</th></tr></thead>
<tbody><tr><td>Data</td></tr></tbody>
<tfoot><tr><td>End</td></tr></tfoot>

<colgroup> & <col> – Style Columns

Used to apply styles to entire columns.

<colgroup>
  <col style="background-color:gray;">
</colgroup>

🔧 Extra Table Attributes

  • colspan: Merges cells horizontally
    • <td colspan="2">Merged</td>
  • rowspan: Merges cells vertically
    • <td rowspan="2">Merged</td>
  • scope="row": Declares header for the row
    • <th scope="row">Raj</th>
  • scope="col": Declares header for the column
    • <th scope="col">Name</th>

🎨 CSS Table Styling - With Code

1. border-collapse

Removes the gap between borders in the table.

table { border-collapse: collapse; }

2. width

Sets the width of the table relative to its container.

table { width: 80%; }

3. margin

Centers the table and adds spacing above and below.

table { margin: 20px auto; }

4. background-color

Applies background color to cells or headers.

th {
  background-color: #003300;
}
tr:nth-child(even) {
  background-color: #111111;
}

5. color

Sets the color of text in header cells.

th { color: #33ff33; }

6. padding

Adds space inside each cell to make content readable.

td, th { padding: 8px; }

7. text-align

Aligns text horizontally inside cells.

td, th { text-align: center; }

8. border

Adds border to each table cell.

td, th { border: 1px solid #00ff00; }

9. nth-child

Adds zebra stripes (alternating row colors).

tr:nth-child(even) { background-color: #111111; }
