Text Tags in HTML


This page will help you understand how to use the <p> and <pre> tags.

The <p> Tag

The <p> tag is used to define a paragraph. It automatically adds some margin and spacing around the text. You can have multiple paragraphs on a page. Here is an example: 

<p>This is my first paragraph.</p>
<p>This is my second paragraph.</p>

Output:

This is my first paragraph.

This is my second paragraph.

The <pre> Tag

The <pre> tag is used for preformatted text. It displays text exactly as it is written in the HTML file, including spaces and line breaks. This is useful when you want to show code or poetry. 

<pre>
This text
  will keep
    its formatting
like this!
</pre>

Output: 

This text
  will keep
    its formatting
like this!
