Lists in HTML


HTML offers three types of lists: unordered, ordered, and description lists. Lists help organize content clearly and cleanly.

1. Unordered List - <ul>

Unordered lists show items with bullets (•).

<ul>
  <li>Milk</li>
  <li>Eggs</li>
  <li>Bread</li>
</ul>

2. Ordered List - <ol>

Ordered lists show items with numbers, letters, or roman numerals.

<ol>
  <li>Wake up</li>
  <li>Brush teeth</li>
  <li>Have breakfast</li>
</ol>

Type Attribute

You can change the numbering style using type:

  • type="1" – Numbers (1, 2, 3)
  • type="A" – Capital letters (A, B, C)
  • type="a" – Lowercase letters (a, b, c)
  • type="I" – Upper Roman numerals (I, II, III)
  • type="i" – Lower Roman numerals (i, ii, iii)
<ol type="A">
  <li>Alpha</li>
  <li>Beta</li>
</ol>

Start Attribute

Use start to begin from a specific number or letter:

<ol start="5">
  <li>Step 5</li>
  <li>Step 6</li>
</ol>

3. Description List - <dl>

A description list contains terms and their descriptions using <dt> and <dd>. <dt> is ued for the term name & <dd> is used for the description of the term.

<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>

  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>

4. Nested Lists

You can place one list inside another to create sublists. The nested list should be the child of (inside) the <li>tags.

<ul>
  <li>Fruits
    <ul>
      <li>Apple</li>
      <li>Banana</li>
    </ul>
  </li>
  <li>Vegetables</li>
</ul>
