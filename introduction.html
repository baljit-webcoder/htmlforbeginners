Introduction to HTML


This page gives you a basic introduction to HTML.

File Extension

An HTML file always has either a .html or a .htm file extension. This allows the browser to recognize and render the site correctly.

HTML Tags

HTML is written using tags. There are two types of tags: self-closing tags that do not have content inside them, and non-self-closing tags that do have content inside them.

These tags are written inside < and >. For example, <h1>Hello</h1> is a heading tag.

If you want to display special characters like < or > on your webpage, you must use HTML entities:

  • To get <, type: &lt;
  • To get >, type: &gt;
  • To get &, type: &amp;

Declaration

Every HTML page starts with a declaration tag. This tag always sits at the top of the page. In HTML5, we use <!DOCTYPE html> to tell the browser that the webpage is written using HTML5.

HTML Tag

All HTML code must be written inside <html> and </html> tags. These tags should be placed just after the declaration.

<head> Tag

The <head> section contains important information about the webpage, such as the title, character encoding, linked stylesheets, and meta tags. It is not displayed on the actual webpage.

<body> Tag

The <body> section contains everything that you see on the webpage. This includes headings, paragraphs, images, links, buttons, and more.

HTML lang Attribute

The lang attribute is added to the <html> tag to specify the language of the page. For example, <html lang="en"> means the content is in English. This helps screen readers and browsers understand the content better.

Here's an example of how to use it:

<!DOCTYPE html>
<html lang="en">
  <head>
    <title>My Page</title>
  </head>
  <body>
    <p>Hello, world!</p>
  </body>
</html>
