Prerequisites

There are no prerequisites needed to learn HTML! You only need a computer or a laptop & a mindset and will to learn & master this beautiful language & start your journey as a web developer.

So I'm ready to learn this language to learn HTML. But what is this & why should I learn this?

If you have such questions you have come to the right place. So let's start with the beginning.

1. What is HTML?
HTML stands for Hyper Text Markup Language. Together with CSS (Cascading Style Sheets) and JS (JavaScript) it is used to make websites on the internet.
2. Why do I need to learn this?
It improves your CV, lets you build websites, unleashes creativity, and is one of the easiest programming languages.
3. Is HTML case-sensitive?
No. HTML is not case-sensitive. But it is a good practice to write tags in lowercase.
4. Can I use HTML without CSS or JS?
Yes, but your websites will be plain. CSS styles it, JS makes it interactive.
5. Is HTML enough to build a full website?
Not really. HTML builds the structure. You also need CSS and JS for full functionality and styling.

Setting up your workflow

To write and view HTML, you need:

  • A Code Editor:
    • Visual Studio Code (Most preferred)
    • Sublime Text (Freemium)
    • Atom (No longer maintained but still downloadable)
    • Brackets
    • VSCodium (Open Source)
  • A Web Browser:
    • Google Chrome (Most useful)
    • Apple Safari (on Mac)
    • Microsoft Edge
    • Mozilla Firefox
    NOTE: Do not use Internet Explorer anymore, if you still use that.

We recommend: Google Chrome + Visual Studio Code on Windows 11, as it is the best combination.

Optional Suggestions

  1. Customize your browser & IDE to your liking.
  2. Create a GitHub account.
  3. Create a folder on your desktop to organize your HTML files.

You’re ready! Take a little break, then move on to writing your first HTML code.
