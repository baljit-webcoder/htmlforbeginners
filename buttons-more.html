Buttons, Line Breaks, Rules & Comments in HTML


1. Buttons - <button> and Hyperlink Style

The <button> tag defines a clickable button. It is mostly used in forms or styled with CSS for actions.

<button>Click Me</button>

HTML buttons are often used for submitting forms:

<form>
  <input type="text" name="name">
  <button type="submit">Submit</button>
</form>

To make a button that works like a link (without JavaScript), you can wrap a <button> inside an <a> tag:

<a href="https://example.com">
  <button>Visit Example</button>
</a>

OR you can style a link to look like a button:

(We will use this after learning CSS)

<a href="https://example.com" class="button">Styled Link Button</a>

2. Line Break - <br>

The <br> tag breaks a line and continues on the next line.

This is line one.<br>
This is line two.

3. Horizontal Rule - <hr>

The <hr> tag draws a horizontal line to divide sections.

<p>Above line</p>
<hr>
<p>Below line</p>

4. Comments - <!-- -->

Use comments to leave notes inside HTML code. They are not shown on the page.

<!-- This is a comment -->
<p>This is visible text</p>
