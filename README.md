# HTML for Beginners

A simple, straight-to-the-point tutorial website made for absolute beginners to learn **HTML** quickly and effectively.  
This project is created and maintained by **[Baljit Panda](https://baljitpanda.rf.gd)** and is completely open-source.

---

## 🌐 Live Website

👉 Visit the site here: [htmlforbeginners.rf.gd](https://htmlforbeginners.rf.gd)

---

## 📁 About the Project

This project contains:
- Beginner-friendly **HTML notes** in website format
- Clean, dark-themed layout (inspired by VS Code)
- Easy-to-read **examples**, **explanations**, and **navigation**
- A downloadable `.zip` version for offline access

**Note:** This project uses **example code only**.  
There are **no real multimedia files (videos, images, or audio)** — only demonstration code for educational purposes.

---

## 📚 Topics Covered

- What is HTML?
- Why learn HTML?
- Getting started with setup
- Writing basic HTML
- Formatting content
- Adding multimedia (theory with sample code)
- More lessons coming soon!

---

## 🛠️ How to Use

You can:
- Read the lessons on the live site
- Download and open the `.zip` folder to read offline
- Use the examples as a reference while learning
- Modify the pages to practice HTML on your own

---

## 💻 Requirements

- A modern browser (Chrome, Firefox, Edge, etc.)
- Any basic code editor (like Visual Studio Code, Sublime Text, or Notepad++)

---

## ⚠️ Note

This is a **learning resource**.  
It does **not** include actual media files — only HTML tags and placeholder examples are used to teach how they work.
