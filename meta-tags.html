Meta Tags in HTML


Meta tags are special tags inside the <head> section of your HTML file. They give information about the page to the browser, search engines, and devices.

✅ Required Meta Tags

1. Charset

Sets the character encoding of your webpage to UTF-8, which supports almost all languages and symbols.

<meta charset="UTF-8">

2. Viewport

Ensures your website looks good on all devices (especially mobile). This is very important for responsive design.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

3. Author

Specifies the author of the webpage.

<meta name="author" content="Baljit Panda">

4. Description

This short sentence tells search engines what your page is about. It may appear in search results.

<meta name="description" content="Your page description here">

5. Keywords

Lists important words that relate to your page. These are used by some search engines to understand your content.

<meta name="keywords" content="html, beginner, learning, tutorial">

6. Robots (optional)

Tells search engines whether they should index your page and follow links. Default is usually "index, follow".

<meta name="robots" content="index, follow">

7. Title

Tells tells the web browser the main title of the page. This is displayed on the title bar of your browser

<title>Page Title</title>
