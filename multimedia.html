Multimedia in HTML


HTML lets you easily embed images, audio, and video content using built-in tags.

1. Image - <img>

<img src="images/sunset.jpg" alt="Sunset View" width="400" height="300">

Attributes:

  • src: Path to the image file
  • alt: Text shown if the image fails to load
  • width, height: Size in pixels

2. Audio - <audio>

<audio controls src="audio/music.mp3"></audio>

Attributes:

  • controls: Shows play/pause UI
  • autoplay: Starts playing automatically
  • loop: Repeats the audio
  • muted: Mutes audio by default
  • preload: Loads file before playing (auto, metadata, or none)
<audio controls autoplay loop muted preload="auto" src="audio/music.mp3"></audio>

3. Video - <video>

<video width="480" height="360" controls src="video/sample.mp4"></video>

Attributes:

  • controls: Displays video controls
  • autoplay: Starts video on load
  • loop: Plays video repeatedly
  • muted: Mutes video by default
  • poster: Preview image before play
  • width, height: Size of the video
  • preload: Loads file before play
<video width="480" height="360" controls autoplay loop muted preload="auto" poster="images/poster.jpg" src="video/sample.mp4"></video>

📁 Understanding File Paths in HTML

When using multimedia in HTML, like images, audio, or video files, the src attribute tells the browser where the file is located. This is called a file path.

Types of Paths:

  • Same folder:
    <img src="photo.jpg">
    → Looks for the file in the same folder as the HTML file.
  • Inside a folder:
    <img src="images/photo.jpg">
    → Goes into the images folder and finds photo.jpg.
  • Going up one folder: ../
    <img src="../images/photo.jpg">
    → Moves up one level and then into the images folder.
  • Going up two folders: ../../
    <img src="../../assets/pic.jpg">
    → Moves up two folders, then into assets.
  • Absolute Path:
    <img src="https://example.com/image.jpg">
    → Loads from a full website URL (online).

💡 Tip:

Keep your folders organized. Create separate folders like images/, audio/, and video/ inside multimedia/ to make paths easier to manage.
