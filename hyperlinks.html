Hyperlinks in HTML


Hyperlinks let users navigate between pages, files, or sections using the <a> tag.

1. Basic Link

<a href="https://example.com">Visit Example.com</a>

2. Open in a New Tab

target="_blank" opens the link in a new browser tab.

<a href="https://example.com" target="_blank">Open in new tab</a>

3. Tooltip with Title

<a href="https://example.com" title="Go to Example">Hover for Tooltip</a>

4. Download a File

<a href="files/sample.pdf" download>Download PDF</a>

5. Email and Phone Links

<a href="mailto:baljit@example.com">Send Email</a>

<a href="tel:+919999999999">Call Me</a>

6. Internal Page Links

<a href="#footer">Go to Footer</a>

...

<footer id="footer">This is the footer section</footer>

7. Safe External Links

Prevent security issues by using rel="noopener noreferrer" with target="_blank".

<a href="https://example.com" target="_blank" rel="noopener noreferrer">Secure Link</a>

8. Link to Local Files

<a href="images/photo.jpg">View Image</a>

<a href="docs/info.html">Open Info Page</a>

9. Nesting Elements

<a href="https://example.com">
  <img src="images/logo.png" alt="Logo">
</a>
