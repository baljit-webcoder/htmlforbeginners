Text Formatting in HTML


This page demonstrates various HTML tags used for formatting text on a webpage.

<b> Tag

The <b> tag makes text bold without implying extra emphasis.

<b>Bold Text</b>

Bold Text

<i> Tag

The <i> tag italicizes text.

<i>Italic Text</i>

Italic Text

<small> Tag

The <small> tag displays text in a smaller size.

<small>Small Text</small>

Small Text

<u> Tag

The <u> tag underlines text.

<u>Underlined Text</u>

Underlined Text

<strong> Tag

The <strong> tag makes text bold and indicates strong importance.

<strong>Strong Text</strong>

Strong Text

<em> Tag

The <em> tag italicizes text and conveys emphasis.

<em>Emphasized Text</em>

Emphasized Text

<sup> Tag

The <sup> tag is used for superscript text.

10<sup>2</sup> gives 10²

102

<sub> Tag

The <sub> tag is used for subscript text.

H<sub>2</sub>O represents water

H2O

<ins> Tag

The <ins> tag indicates inserted (added) text, often underlined.

<ins>Inserted Text</ins>

Inserted Text

<del> Tag

The <del> tag represents deleted (removed) text, typically shown with a strikethrough.

<del>Deleted Text</del>

Deleted Text

<mark> Tag

The <mark> tag highlights text, usually with a yellow background by default (but styled here according to our CSS).

<mark>Marked Text</mark>

Marked Text

<big> Tag

The <big> tag makes text slightly larger (note that it is deprecated in HTML5, but still works in many browsers).

<big>Big Text</big>

Big Text

<s> Tag

The <s> tag indicates text that is no longer accurate or relevant, usually displayed with a line through it.

<s>Strikethrough Text</s>

Strikethrough Text
