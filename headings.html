HTML Headings


Headings are used to define the structure and titles of a web page's content. HTML provides 6 levels of headings, from <h1> to <h6>.

Importance of Headings

Headings help organize your content and improve readability. They also play an important role in SEO (Search Engine Optimization), making your content easier to understand for search engines.

Syntax

The syntax of a heading is simple. For example, to create a main heading, you write: 

<h1>This is a main heading</h1>

Examples

Here’s how all six heading levels look:

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

Best Practices

  • Use only one <h1> tag per page (usually as the main title).
  • Follow a logical structure: h1 → h2 → h3, and so on.
  • Don’t skip levels just for styling; use CSS to style your headings instead.
