Exercise 1 – Practical HTML Task


Solve the following tasks using everything you’ve learned so far. These are real-life tasks you’ll often do when building websites.

1️⃣ Text Formatting

Write a sentence for each of these tags: <b>, <i>, <small>, <u>, <strong>, <em>, <sup>, <sub>, <ins>, <del>, <mark>, <big>, <s>

2️⃣ Hyperlinks

Add the following:

  • Internal link with target and title
  • External link with target and title
  • Email link
  • Anchor link within the same page

3️⃣ Headings

Use all six HTML headings (<h1> to <h6>) with different phrases in each.

4️⃣ Multimedia

Use <img>, <audio>, and <video> — all with all available attributes in each (e.g. controls, autoplay, loop, muted, poster, alt, width, height, etc.).

5️⃣ Link to Solution

Add a button at the end of the page that links to your solution-1.html.

6️⃣ Lists

Create:

  • An unordered list
  • A description list
  • An ordered list using type (e.g. type="a" or type="I")

7️⃣ Daily Routine Table

Create a table of your daily routine using:

  • <table>, <tr>, <th>, <td>
  • <caption>, <thead>, <tbody>, <tfoot>, <colgroup>
  • Attributes like colspan, rowspan, scope

Style the table with internal CSS inside the <style> tag.

8️⃣ GitHub Pages

Create a GitHub repository, push your code there, and deploy it using GitHub Pages.
